2 Основание для разработки
Задание на проектирование в рамках изучения МДК 02.01 «Технология разработки программного обеспечения» по специальности 09.02.07 «Информационные системы и программирование», факультет СПО №12 СПБГУАП
3 Назначение разработки:
Создание обучающей программы для начинающих художников и дизайнеров на тему "Теория цвета".
Описание функциональности продукта:
− Возможности для смешивания различных оттенков.
− Возможность создания изображений с использованием выбранных оттенков.
− Поддержка проверки знаний теории цвета через тесты и викторины.
4 Требования к программе или программному изделию
4.1 Требования к функциональным характеристикам
− Программа должна обеспечивать возможность выполнения следующих функций:
− Смешивание цветов для получения уникальных оттенков с возможностью их сохранения в персональные палитры пользователя.
− Рисование с использованием цветов из созданных палитр.
− Сохранение созданных изображений в различных форматах.
− Импорт изображений для редактирования и применения цветов из палитр.
− Регистрация пользователя
− Вход и выход из аккаунта 
4.1.2. Исходные данные: 
− Для редактирования изображения: исходное изображение, путь к нему, а также разрешение и формат файла.  
− Данные пользователя: имя, электронная почта, информация о регистрации
− Данные о палитрах пользователя, включая названия палитр, цветовые коды сохраненных оттенков. 
4.1.3 Результаты: 
− Результат пройденного тестирования, включая общее количество правильных и неправильных ответов.  
− Созданные оттенки и палитры с присвоенными им названиями.
− Итоги проверки навыков классификации оттенков, включая оценку уровня знаний и областей для улучшения.  
4.2 Требования к надежности 
− Предусмотреть контроль вводимой информации. 
− Предусмотреть блокировку некорректных действий пользователя при работе с системой. 
− Обеспечить целостность хранимой информации.
